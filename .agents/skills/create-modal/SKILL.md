# Skill: Create Modal

## Workflow

1. Find similar modal
2. Reuse existing dialog/modal components
3. Match spacing/layout patterns
4. Match existing button behavior
5. Reuse validation patterns
6. Keep modal logic simple
7. Preserve accessibility patterns

## Forbidden

- Do not create new modal systems
- Avoid custom modal state systems
